How ratings work

Each hour gets one of five ratings using a wind-gate formula. Sub-scores are 0–1.

Wave: 0.05 if < ½ ideal-min (flat) → ramps to 1.0 in the ideal range → drops if too big.
Period: 0 if below min → 0.5–1.0 between min and good → 1.0 above.
Wind: 1.0 light (≤10 mph) → drops to 0.05 blown out (≥25 mph). Offshore +0.15, onshore ×0.6.

Wind-gate scoring:
base = (w_wave × wave + w_period × period) / (w_wave + w_period)
if base < 0.65 → score = base (wind ignored)
else → score = base × (0.6 + 0.4 × wind)

Poor < 0.20 Fair < 0.40 OK < 0.60 Good < 0.80 Epic ≥ 0.80
